# Wijkagent Dashboard

## Over het project
De politie heeft behoefte aan een applicatie die wijkagenten kunnen raadplegen om te zien wat er in de wijk speelt.  
Voorvallen worden gecategoriseerd en vastgelegd in een database.  

De wijkagent wil via deze applicatie een kaart kunnen bekijken waarop delicten zichtbaar zijn die binnen een bepaalde periode hebben plaatsgevonden.  
Daarnaast wil de politie de beschikbare data verrijken met informatie van sociale media, zoals Twitterberichten die rond het tijdstip en de locatie van een delict zijn verzonden.  

Het doel van het systeem is om meer inzicht te krijgen in patronen, hotspots en trends binnen de wijk.
